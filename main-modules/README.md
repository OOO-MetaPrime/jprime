# Состав блоков

| Блок               | Описание                                                             |
|--------------------|----------------------------------------------------------------------|
| core               | Базовые модули                                                       |
| imex               | Модули универсальной утилиты импорта файлов                          |
| meta               | Модули, связанные с метаописанием                                    |
| schedule           | Настройки планировщика                                               |
| security           | Настройки доступа RBAC и ABAC                                        |
| test               | Модули тестирования                                                  |
| utils              | Модули утилит                                                        |
